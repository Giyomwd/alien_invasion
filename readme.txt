介绍：
游戏方式：左右键移动，空格子弹，Q键退出，点击Play按钮开始游戏。
开发思路：
alien_invasion_screen.py
   主程序，整体游戏循环
ship.py
    飞船的渲染，位置的初始化
alien.py
     外星人的渲染，screen中的展示，位置的初始化
game_function.py
     本项目中最重要的工具类，负责主程序实现的具体过程函数
bullet.py
      子弹在screen中的渲染
button.py
      Play键的渲染
game_stats.py
     跟踪游戏的统计信息：分数，最高分数，生命条数等
setting.py
     子弹，飞船，外星人具体行为规则
scoreboard.py
      等级，最高分，分数，飞船生命条数等在screen上的渲染
